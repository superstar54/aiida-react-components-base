
Welcome to AiiDA React components base documentation!
========================================================
AiiDA React components base is a collection of reusable React components designed to work with AiiDA, a powerful computational materials science framework. This library provides a set of components that can be used to build user interfaces for AiiDA applications, enhancing the user experience and streamlining the development process.
